# 武器と盾の拡張

Minecraftの戦闘をもっと奥深く、楽しくするために、このサーバーには **Spartan Weaponry** と **Spartan Shields** というModが導入されています。

## 多彩な「武器種」とその特性

通常の剣や斧に加えて、槍やハルバード、メイスなど、個性豊かな武器がたくさん追加されています。武器種によってそれぞれ異なる特徴（特性）を持っています。

### 代表的な武器と特性の例

- **ダガー**: 攻撃力は低いですが、攻撃速度が非常に速いです。また、背後から攻撃すると大ダメージを与えるバックスタブ特性があります。
- **槍 / パイク**: リーチ（攻撃距離）が通常の武器よりも長いため、敵の攻撃が届かない位置から一方的に攻撃できます。
- **メイス / ウォーハンマー**: 攻撃速度は遅いですが、敵の防具（アーマー値）を一部貫通してダメージを与える特性があり、硬い敵に有効です。
- **ハルバード**: 高いダメージと長いリーチを併せ持ちますが、盾と同時に使いにくい（両手持ち推奨）というデメリットがあります。
- **ブーメラン**: 投げると手元に戻ってくる遠距離武器です。

## Spartan Shields による盾の強化

バニラの盾だけでなく、さまざまな素材で作れる新しい盾が追加されています。

1. **素材ごとの性能**: 木、鉄、ダイヤ、ネザライトなど、使った素材に応じて盾の耐久値や、攻撃を防いだ際のダメージカット率・ノックバック耐性が変化します。
2. **タワーシールド（Tower Shield）**: 通常の盾よりも巨大で動きが少し遅くなりますが、より強力な防御性能を誇ります。
3. **シールドバッシュ**: 追加された盾には、防御しながら敵を押し返すシールドバッシュの機能がついているものもあります。

プレイスタイルに合わせて武器と盾を組み合わせ、自分だけの戦闘スタイルを見つけましょう！
