# 工業(Create)の進め方

Create Modは、歯車や動力を使って様々な機械や自動化ラインを作るためのModです。

## 最初の一歩：安山岩合金

Createで機械を作るには、まず**安山岩合金（Andesite Alloy）**が必要です。
これは、以下の素材をクラフトして作成します。

- 安山岩（Andesite）
- 鉄インゴット（Iron Ingot）または 亜鉛インゴット（Zinc Ingot）

```xml
  [安山岩] [鉄 or 亜鉛]
```

## 動力源の確保

最初は**水車（Water Wheel）**を作るのがおすすめです。
設置した水車に水流を当てることで回転動力が生まれます。

- **小さな水車**: 作りやすく、設置しやすい。
- **大きな水車**: コストはかかりますが、より多くの動力を生み出せます。

## 目安となる機械

1. **メカニカルプレス（Mechanical Press）**: 鉄インゴットを鉄板に加工します。
2. **ミキサー（Mechanical Mixer）**: ブレイズバーナーと組み合わせて合金を作ります。

頑張って自動化拠点を構築しましょう！
